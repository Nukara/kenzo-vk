Очередь загрузок
==========================

Очередь разделена на две части:
а) активные загрузки,
б) собственно очередь из которой загрузки стартуют, если есть доступные слоты
для скачивания.

В очереди регистрируются все активные в данный момент загрузки браузера, таким
образом они тоже участвуют в логике работы очереди.

Например, если количество слотов активных загрузок — 3, в очередь тправлено 10 файлов
(например аудиозаписей), то 3 из них сразу же стартуют, а остальные 7 ждут своей очереди,
пока слоты не освободятся. Если в этот момент браузер стартует…
